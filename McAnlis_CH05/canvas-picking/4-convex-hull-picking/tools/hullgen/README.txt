For this source code to work, you'll need to grab a copy of the FreeImage library : http://freeimage.sourceforge.net/
and free image plus : http://freeimage.sourceforge.net/fip/

Place files to generate this file structure:
tools/hullgen/FreeImage.h
tools/hullgen/FreeImage.lib
tools/hullgen/contrib/FreeImagePlus/FreeImagePlus.dll
tools/hullgen/contrib/FreeImagePlus/FreeImagePlus.h
tools/hullgen/contrib/FreeImagePlus/FreeImagePlus.lib
tools/bin/FreeImagePlus.dll
